# lita-ping

**lita-ping** is a handler for [Lita](https://github.com/jimmycuadra/lita) that gives you back a response if the bot is listening.

## Installation

Add lita-ping to your Lita instance's Gemfile:

``` ruby
gem "lita-ping"
```

## Configuration

None.  Just use it.

## Usage

```
user1> lita ping
Lita> Pong
```
